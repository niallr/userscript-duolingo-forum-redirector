# userscript-duolingo-forum-redirector


## Duolingo Forum Redirector
A lightweight userscript that automatically redirects dead Duolingo Forum links (duolingo.com/comment/*) to the community-maintained Duolingo Forum Archive at duolingo.hobune.stream.

Since Duolingo officially deactivated their forums, years of valuable grammar discussions and community explanations became inaccessible. This script restores those links by pointing them to the most comprehensive static archive available.
